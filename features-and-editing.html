Feature 01

Inline Text
Editing

Click any text element with the data-editable attribute and it becomes a live content-editable field. There are no forms, no modals, no separate editing views.

Changes are made directly on the rendered page, so you always see exactly what your visitors will see. Headings, paragraphs, span elements, list items — anything can be made editable with a single attribute.

Minimal setup required:

<p data-editable>Your text here</p>
Feature 02

Image
Replacement

Any <img> with the data-editable-image attribute becomes clickable. Clicking it opens your native file picker.

When you select a new image, Webby uploads it directly to your GitHub repository using the Contents API and updates the src attribute on the page in real time. No manual uploading, no path editing.

<img src="./assets/hero.jpg" data-editable-image alt="..." />
Feature 03

AI Section
Builder

Hover between any two sections on the page and a glowing + Add Section button appears. Click it, describe what you want in plain English, and Claude generates a complete, styled HTML section matching your site's aesthetic.

The generated section is inserted directly into the page with all the correct data-zone, data-editable, and data-editable-image attributes already in place — ready to edit immediately.

Example prompts that work beautifully:

  • → "A pricing table with three tiers"
  • → "A team section with four photo cards"
  • → "An FAQ accordion with five questions"
  • → "A contact form with name, email, and message"

Features 04 & 05

Theme & Publishing

🎨

Live Theme Editor

Click the Theme button in the toolbar to open the theme panel. Adjust your CSS custom properties — colours, fonts, spacing — and watch the entire page update in real time.

  • ✦   Colour palette (primary, secondary, accent, background)
  • ✦   Typography (heading and body font families)
  • ✦   Spacing scale (xs through xl)
  • ✦   Border radius and shadow presets
🚀

One-Click Publishing

Click Publish in the toolbar. Webby serialises the current page, strips all editor code, credentials, and edit-mode attributes, then calls the GitHub Contents API to write the file directly to your repository.

  • ✦   No git, no terminal, no syncing
  • ✦   GitHub Actions picks up the push automatically
  • ✦   Live on GitHub Pages in ~60 seconds
  • ✦   Zero editor code in the deployed output

Reference

HTML Data
Attributes

Attribute Applied To Purpose
data-zone <section> Marks a top-level editable section. Value is a unique slug, e.g. "hero".
data-zone-label <section> Human-readable label shown in confirmations (e.g. delete section), e.g. "Hero".
data-editable Any text element Activates contenteditable on click. Works on h1–h6, p, span, li, and more.
data-editable-image <img> Click-to-replace: opens a file picker and uploads the new image to GitHub.
